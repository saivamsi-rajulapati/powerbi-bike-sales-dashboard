# 📊 Bike Sales Analytics Dashboard – Power BI

This repository contains a Power BI dashboard created to analyze bike sales data using interactive visualizations and clear data storytelling. The project focuses on transforming raw sales data into meaningful insights that help understand customer behavior, purchasing trends, and overall sales performance.

The dashboard was built using Microsoft Power BI with data modeling techniques and DAX calculations to create dynamic visuals and filters. It allows users to explore sales information in a simple and user-friendly way, making it easier to identify patterns and key performance indicators.

Through this project, I practiced real-world data analytics skills such as cleaning datasets, designing dashboards, and presenting insights visually. The aim was to build a professional and structured report that demonstrates practical knowledge in Power BI and data visualization.

## 📂 Files Included
- bi dashboard.pbix – Main Power BI dashboard
- BikeSales-DB.xlsx – Dataset used for analysis

## 🛠 Tools Used
- Microsoft Power BI
- Data Modeling
- DAX
- Data Visualization
